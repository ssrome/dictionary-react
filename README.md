# Dictionary React App

A basic to-do app that performs CRUD operations locally. A singular list will be saved to local storage.

## Installation

```
npm install
```

## Technologies

- React
- Bootstrap
- Axios

You can see this project [here](https://cocky-brown-1edec7.netlify.app/)
